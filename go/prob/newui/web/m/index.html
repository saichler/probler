Platform Components

Focused services with single responsibilities, connected through explicit runtime behavior.

l8services
Model-agnostic services that adapt to any data structure without code changes.
l8bus
Virtual network overlay with routing, health signaling, and built-in discovery.
l8collector
Multi-protocol data collection with horizontal scaling across large infrastructure.
l8parser
Model-agnostic parsing and transformation for normalized runtime objects.
l8orm
Object-relational mapping for durable storage of modeled infrastructure state.
l8topology
Network topology with geographic positioning and interactive visualization.
