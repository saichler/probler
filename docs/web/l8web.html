Core Features

Built for performance, security, and scalability in distributed environments

RESTful API Server

Full HTTP REST server with native Protocol Buffers integration, supporting all standard HTTP methods with automatic serialization.

Reverse Proxy

Multi-domain reverse proxy with SSL termination, load balancing, WebSocket support, and intelligent request routing.

Service Discovery

Automatic service registration and discovery with Layer 8 integration, enabling seamless distributed communication.

TLS/HTTPS Security

Built-in SSL/TLS with automatic certificate generation, bearer token authentication, and custom CA support.

Plugin System

Dynamic plugin loading with hot-reload capability, enabling extensible service implementations without restarts.

Multi-cast Communication

Integration with Layer 8's proximity-based routing for efficient distributed message passing.

Quick Start Examples

Creating a Web Server

import ( "github.com/saichler/l8web/go/web/server" ) // Configure the server with enhanced features config := &server.RestServerConfig{ Host: "localhost", Port: 8080, Authentication: true, // Enable auth CertName: "server", // Auto-generate SSL cert Prefix: "/api/v1/", DevMode: true, // Enable hot-reload } // Create and start the server srv, err := server.NewRestServer(config) if err != nil { log.Fatal(err) } // Register middleware srv.Use(LoggingMiddleware) srv.Use(RateLimitMiddleware) // Start the server go srv.Start()

Setting up Reverse Proxy

import ( "github.com/saichler/l8web/go/web/proxy" ) // Configure multi-domain proxy proxyConfig := &proxy.Config{ Domains: map[string]proxy.DomainConfig{ "api.example.com": { Backend: "http://localhost:8080", SSL: true, Cache: true, }, "app.example.com": { Backend: "http://localhost:3000", SSL: true, WSSupport: true, // WebSocket support }, }, LoadBalancing: proxy.RoundRobin, HealthCheck: true, } // Create and start proxy proxy, err := proxy.NewProxy(proxyConfig) if err != nil { log.Fatal(err) } // Start proxy server proxy.ListenAndServe(":443")

Creating a Web Client

import ( "github.com/saichler/l8web/go/web/client" ) // Configure the client with advanced features config := &client.RestClientConfig{ Host: "api.example.com", Port: 443, Https: true, CertFileName: "ca.crt", TokenRequired: true, Token: "Bearer " + getAuthToken(), Prefix: "/api/v1/", Timeout: 30 * time.Second, MaxRetries: 3, } // Create the client with connection pooling restClient, err := client.NewRestClient(config, resources) if err != nil { log.Fatal(err) } // Add request interceptor restClient.AddInterceptor(func(req *http.Request) { req.Header.Set("X-Request-ID", generateRequestID()) }) // Make requests with automatic retry response, err := restClient.GET("/users", User{}, "", "", nil)

Service Registration & Discovery

// Register a web service handler webNic.Resources().Services().RegisterServiceHandlerType( &server.WebService{ Name: "UserService", Version: "1.0.0", Endpoints: []string{"/users", "/auth"}, HealthCheck: "/health", }, ) // Activate the web service with auto-discovery service, err := webNic.Resources().Services().Activate( server.ServiceTypeName, ifs.WebService, 0, // Auto-assign port webNic.Resources(), webNic, srv, ) // Service is now discoverable via Layer 8 log.Printf("Service registered: %s at %s", service.Name(), service.Endpoint()) // Discover other services services := webNic.DiscoverServices("UserService") for _, svc := range services { log.Printf("Found service: %s", svc.Endpoint()) }

Getting Started

📦 Installation

go get github.com/saichler/l8web/go # With all dependencies go mod download

🧪 Testing

# Run full test suite ./test.sh # Run with coverage go test -coverprofile=cover.out ./... go tool cover -html=cover.out

🚀 Quick Deploy

# Build the server go build -o l8web cmd/server/main.go # Run with config ./l8web --config server.yaml # Docker deployment docker build -t l8web . docker run -p 8080:8080 l8web

📋 Requirements

  • Go 1.23.8 or higher
  • Layer 8 Framework Core
  • Protocol Buffers 3.0+
  • Docker (optional)
  • SSL certificates (auto-generated if not provided)

🔧 Configuration

# server.yaml server: host: 0.0.0.0 port: 8080 ssl: enabled: true cert: server.crt key: server.key auth: enabled: true token_expiry: 24h layer8: discovery: true multicast: true
