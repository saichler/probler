🚀 Quick Start

Get Probler up and running in minutes with our simple deployment process

1

Clone Repository

Get the latest version of Probler from GitHub

# Clone the repository git clone https://github.com/saichler/probler.git cd probler
2

Build Services

Build all microservices using the provided script

# Build all services cd go && ./build-images.sh
3

Deploy Platform

Choose your deployment method

# Docker Compose (Development) docker-compose up -d # Kubernetes (Production) cd k8s && ./apply-all.sh
4

Access Dashboard

Open the Network Operations Center

# Local access open https://localhost:443/probler/ # Default credentials: admin / admin

Prerequisites

Go 1.23.8+ Docker & Docker Compose Kubernetes cluster PostgreSQL database
