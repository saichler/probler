Layer 8 Distributed ORM

Comprehensive distributed Object-Relational Mapping service built on the Layer8 networking framework. Designed for modern distributed applications with advanced querying, PostgreSQL support, and automatic object conversion.

Get Started

Key Features

Distributed Architecture

Horizontally and vertically scalable ORM service with service mesh integration designed for high-performance distributed systems.

Layer8 Integration

Built on the Layer8 networking framework for reliable distributed communication, service discovery, and management.

🐘

PostgreSQL Support

Native PostgreSQL database integration with optimized query execution and connection management.

Object Conversion

Automatic conversion between Go objects and relational data structures with advanced type safety and reflection.

Protocol Buffers

Uses protobuf for efficient data serialization and cross-platform compatibility with optimized performance.

Transaction Support

Built-in transaction management with replication capabilities, ACID compliance, and timeout support.

Advanced Querying

Sophisticated query builder with filtering, timeout support, and query optimization for complex operations.

Service Bus Integration

Built-in support for l8bus messaging, event-driven architecture, and service mesh communication.

Architecture Overview

Core Components

  • ORM Interface (IORM): Core interface for read/write operations
  • Convert Service: Handles object-to-relational data conversion
  • Persist Service: Manages database persistence operations
  • Statement Builder: SQL statement generation and query building
  • Web Service Layer: REST API endpoints for external access

Installation & Setup

1. Install Package

go get github.com/saichler/l8orm/go

2. Setup PostgreSQL

# Start PostgreSQL
./scripts/start-postgres.sh

# Stop PostgreSQL
./scripts/stop-postgres.sh

3. Generate Proto Bindings

cd proto
./make-bindings.sh

4. Run Tests

cd go
./test.sh

Basic Usage

package main

import (
    "github.com/saichler/l8orm/go/orm/common"
    "github.com/saichler/l8orm/go/orm/persist"
)

func main() {
    // Initialize ORM service
    ormService := &persist.OrmService{}

    // Database operations with timeout
    data, err := orm.Read(query)
    err = orm.Write(relationalData)

    // Object operations with advanced filtering
    elements := orm.ReadObjects(query, resources)
    err = orm.WriteObjects(elements, resources)
}

API Reference

Core IORM Interface

type IORM interface {
    Read(ifs.IQuery) (*types.RelationalData, error)
    Write(*types.RelationalData) error
    ReadObjects(ifs.IQuery, ifs.IResources) ifs.IElements
    WriteObjects(ifs.IElements, ifs.IResources) error
    Close() error
}

Data Structures

The ORM uses Protocol Buffer definitions for data structures:

  • RelationalData: Root container for table data
  • Table: Represents database tables with columns and rows
  • Row: Individual record with column values
  • InstanceRows: Grouped rows by instance
  • AttributeRows: Grouped rows by attribute
